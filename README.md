# Network Pro — Demo

Professional networking app that helps teams turn every event into pipeline.

## Quick Start

### Mac / Linux
```bash
./start-demo.sh
```

### Windows
```
start-demo.bat
```

### Manual
If the scripts don't work, run:
```bash
npx serve .
```
Then open http://localhost:3000?demo=true in your browser.

> **Note:** The `?demo=true` parameter loads the app with sample data so you can explore all features without any account setup.

## What You'll See

- **Contact List** — 15+ sample contacts with AI-enriched profiles
- **Smart Search** — Find contacts by name, company, or context
- **Follow-up Reminders** — Automated nudges to reconnect at the right time
- **Contact Enrichment** — LinkedIn, company info, and social profiles auto-filled
- **Team Workspace** — Shared contact pool, admin dashboard, team analytics (Pro Teams)
